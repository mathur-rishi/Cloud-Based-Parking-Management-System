This system integrates the hardware and software fields to provide parking management solution in India. It is developed by the students of GHRCE, Nagpur.

Info

This website acts as web interface for clients for booking parking slots.

Proposed System

This system is based on IoT & cloud computing in which customer can book their parking slot on our website which is integrated with our hardware. Hardware part is consist of two modules entrance and parking. It is a optimum integration of hardware and software to provide transparent and flexible booking service.

Features

This system provides several features for parking space owners in India and their clients.

Parking Management

Utilizes both implemented hardware at parking and cloud data to manage parking layouts.

In-Slot Tracking

Tracks the entered vehicle at the service point to its respective parking space slot.

Token Bookings

Provides consumer flexibility to book online and later at the entry pay via offline means.

Online Reservations

Can provide variety of option for consumers to pay online and confirm their parking reservation.

Cloud Integration

Utilizes online subscription based database hosted in the Google Cloud service named Firebase.

Maps Integration

Utilizes API in Google Cloud service for integrating the Google Maps with web interfaces.

Responsive Interface

Supports every modern display size to provide flaw-less experience in accessing contents.

Analytical Dashboard

Provides extremely detailed local dashboard at the service point to get overview of parking space. 

FASTag Payments

Can be integrated with the secured server of banks for quick payment via vesicle's RFID tag.

Open Source

Has potential to get multiple contributors, Its repo. will be soon available on Github.
